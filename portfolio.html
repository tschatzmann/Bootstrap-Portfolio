LIRI BOT

LIRI is like iPhone's SIRI. However, while SIRI is a Speech Interpretation and Recognition Interface, LIRI is a Language Interpretation and Recognition Interface. LIRI will be a command line node app that takes in parameters and gives you back data.

  • concert-thie
  • spotify-this-song
  • movie-this
  • Do-what-it-says

After choosing the command LIRI prompts you enter a search value.

  • Enter a band
  • Enter a song
  • movie name
  • Do not enter anything. Search value is stored in the file random.txt

Train Scheduler

This page is simulates a train schedule application. The user enters the train name, final destination, start time and frequency to the current train schedule. The page displays the train schedule in a table. User can delete a row of train information from schedule by clicking the delete button. The page was built using Bootstrap, Javascript, Moment.js, JQuery and Firebase.
